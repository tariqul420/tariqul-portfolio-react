## 💡 **How to Run Locally**

1. **Clone the repository**:

   ```bash
   git clone https://github.com/tariqul420/tariqul-portfolio.git

   ```

2. **Navigate to the project directory**:

   ```bash
   git clone https://github.com/tariqul420/tariqul-portfolio.git

   ```

3. **Install dependencies**:

   ```bash
   npm install

   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
